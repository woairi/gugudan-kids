# 실기기 QA 체크리스트 (모바일)

배포: https://gugudan-kids.vercel.app/

## 기기/브라우저
- Android Chrome
- iPhone Safari

## 공통
- [ ] 버튼 터치 영역 충분(오터치 적음)
- [ ] 글자/숫자 가독성 OK
- [ ] 오답 피드백이 위압적이지 않음

## Learn
- [ ] 단 선택 0~9 정상
- [ ] 랜덤 한 줄 복습(뽑기) 동작
- [ ] Quiz 브릿지 팝업 동작

## Quiz
- [ ] 단 선택 후 시작
- [ ] 즉시 채점/다음 문제
- [ ] 약한 문제 모드 동작(데이터 없을 때 랜덤 전환)
- [ ] 난이도 0~12 설정 시 ×10~×12 출제 확인
- [ ] 문제 수 10/20 반영 확인
- [ ] 퀴즈 진행 중 홈 이동 시 확인창
- [ ] 이어서 하기 복구(문제 번호/맞은 개수/오답 목록)

## Result
- [ ] 다시 풀어볼 문제 표시
- [ ] 최근 기록 표시

## Collection
- [ ] 잠김/획득 표시 직관적
- [ ] 배지 언락 정상(첫퀴즈/만점/단 마스터)

## Parents
- [ ] 사운드 ON/OFF 저장
- [ ] 모두 지우기(2초 꾹) 동작 + 이후 홈 상태 반영
